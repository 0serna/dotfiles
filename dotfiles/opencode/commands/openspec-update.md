---
description: Update OpenSpec to the latest version
---

## Task

1. Run `npm install -g @fission-ai/openspec@latest`
2. Run `openspec init --tools opencode`
3. The command will create a `.opencode` directory with updated commands and skills.
4. Copy or replace those commands and skills into `dotfiles/opencode/commands` and `dotfiles/opencode/skills`.
5. Delete the `.opencode` directory when finished.
